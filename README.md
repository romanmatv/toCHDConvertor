# toCHDConvertor
Simple cue/gdi to chd convertor based on chdman cli

* Scan sub-dirs
* Skip exist chd
* Export chd to up-folder

Application icon download from flaticon.com

## Подробнее
Простой конвертор cue/gdi в chd. Сканирование всех подпапок в выбранной папке. Автопропуск уже существующих chd. Помещение chd как рядом так и на папку выше исходного файла.
